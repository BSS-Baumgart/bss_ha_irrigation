# Irrigation BSS

Short intro for Home Assistant Add-on Store.

Irrigation BSS is an advanced garden watering addon for Home Assistant.
It lets you manage sections, valves, schedules and blocking sensors from one UI panel.

## Stable installation

Add this repository in Home Assistant Add-on Store:

https://github.com/BSS-Baumgart/bss_ha_irrigation
